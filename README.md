# RentIQ

A rental management platform for landlords and tenants to manage properties, payments, and maintenance requests.

## Features

- Property and tenant management
- Payment tracking and processing
- Maintenance request system
- Analytics dashboard
- Document storage

## Tech Stack

**Frontend:** React, TypeScript, CSS  
**Backend:** Node.js, Express, TypeScript  
**Tools:** Docker

## Installation

```bash
git clone https://github.com/angarak994/rentIQ_.git
cd rentIQ_
npm install
npm run dev
```

## Usage

Landlords can manage properties and tenants from the dashboard. Tenants can view their lease, pay rent, and submit maintenance requests.

## Project Structure

```
rentIQ_/
├── backend/          # Node.js API
├── frontend/         # React app
└── docker-compose.yml
```
